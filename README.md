# BDR Repository

Welcome! This repository is a central hub for my Business Development Representative work, containing notes, templates, and learning resources.

## 📂 Repository Structure

```
├── /notes          # Daily notes, observations, and insights
├── /templates      # Email templates, call scripts, pitch decks
├── /learning       # Resources, articles, and skill development
├── /deals          # Deal tracking and case studies
└── /resources      # Tools, documentation, and reference materials
```
